# Car Sale Page with React
Welcome to the Car Sale Page repository! This project is a simple and stylish web page for showcasing cars available for sale, built using React with Vite as the bundler. The user interface is enhanced with Tailwind CSS, providing a clean and responsive design.

## Features:
### React Router:
Seamless navigation between different sections of the application for a smooth user experience.

### React-responsive-modal:
Interactive and visually appealing modals for displaying additional details or actions related to each car.

### SweetAlert2:
Delightful pop-up alerts to enhance user interactions and provide feedback on important actions.

## Technologies Used:
### React:
A powerful JavaScript library for building user interfaces.

### Vite:
A fast build tool that significantly speeds up the development process by providing a lightning-fast dev server and optimized production builds.

### Tailwind CSS:
A utility-first CSS framework that makes styling components a breeze and ensures a consistent design across the entire application.

### React-responsive-modal:
A lightweight and customizable modal library for React.

### SweetAlert2:
A beautiful, responsive, and customizable alert library for better user feedback.
